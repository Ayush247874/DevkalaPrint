# DevkalaPrint

DevkalaPrint is a full-stack printing service website built using React, Node.js, Express, MongoDB, Cloudinary, and Razorpay.

---

# Features

- User Signup/Login
- Upload Front & Back Images
- Order Management
- Admin Dashboard
- Razorpay Payment Gateway
- UPI / QR Payment Support
- Cloudinary Image Upload
- Responsive Design

---

# Tech Stack

## Frontend
- React

- Axios
- Framer Motion

## Backend
- Node.js

# Installation

## Clone Repository

```bash
git clone YOUR_GITHUB_LINK
```

---

# Backend Setup

```bash
cd backend
npm install
npm run dev
```

---

# Frontend Setup

```bash
cd frontend
npm install
npm run dev
```

---

# Environment Variables

Create `.env` file inside backend folder:

```env
MONGO_URI=YOUR_MONGO_URI

CLOUDINARY_CLOUD_NAME=YOUR_CLOUD_NAME
CLOUDINARY_API_KEY=YOUR_API_KEY
CLOUDINARY_API_SECRET=YOUR_API_SECRET

RAZORPAY_KEY_ID=YOUR_KEY_ID
RAZORPAY_KEY_SECRET=YOUR_SECRET_KEY
```
